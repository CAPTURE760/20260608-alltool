# DevKit - 在线开发者工具集

轻量、快速、无广告的在线工具网站。

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS

## 开发

```bash
npm install
npm run dev
```

访问 http://localhost:3000

## 功能列表

### MVP 版本（当前）

- [x] JSON 格式化/压缩
- [x] Base64 编解码
- [x] URL 编解码
- [x] Hash 计算（MD5/SHA1/SHA256）
- [x] 时间戳转换
- [x] UUID 生成
- [x] 字数统计
- [x] 正则测试
- [x] 代码格式化
- [x] 二维码生成
- [x] 图片压缩
- [x] 图片格式转换

## 特点

- 所有处理在浏览器本地完成，不上传任何数据
- 无广告、无追踪
- 响应式设计，支持移动端
